# ⚡ Voltify – Full Stack eCommerce Platform

> A modern full-stack eCommerce web application delivering a seamless and scalable online shopping experience.

---

## 🚀 Features

* 🛍️ Product browsing, search & filtering
* 🛒 Add to cart & smooth checkout
* 🔐 User authentication (JWT based)
* 📦 Order management system
* 📊 Admin dashboard for full control
* 📱 Secure Payment Integration

---

## 🛠️ Tech Stack

**Frontend:** React.js
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Auth:** JWT (JSON Web Tokens),Bcrypt,token,Random String
**Payment:** Stripe
**Images:** Cloudinary
**Dashboard:** Power BI

---

## 📂 Project Structure

/FrontendUser → Frontend User (React)
/FrontendAdmin → Frontend Admin (React)
/Backend → Backend (Node + Express)
/AdminDashboard → Dashboard (Power BI)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/voltify.git
cd voltify
```

### 2️⃣ Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file in `/server`:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
PORT=5000
```

### 4️⃣ Run the project

```bash
# Run backend
cd server
npm run dev

# Run frontend
cd client
npm start
```
